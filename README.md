# dreamy-
An original full-stack task visualising and managing application with a React frontend and a secure Node.js, Express, and MongoDB backend, with JWT-based authentication
## Dreamy

Dreamy is a **full-stack MERN task visualising management application** built to feel minimal, and distraction-free while handling real-world functionality like authentication, data persistence, and production deployment.

It combines a modern React frontend with a secure Node.js and Express backend powered by MongoDB.

---

## Features

* User authentication (Sign up & Login)
* Create, update, and manage tasks
* Time and date tracking for tasks
* Clean, minimal UI
* Secure REST API with JWT-based authentication
  
---

## Tech Stack (MERN)
### Frontend
